# MediSummary AI

MediSummary AI is a C#/.NET web application that uses AI to summarize pasted document text into a simpler summary with key points and action items.

## Features

- Paste document text into a web form
- Generate an AI-assisted summary
- Display key points and action items
- Built with ASP.NET Core MVC
- Uses the OpenAI API

## Tech Stack

- C#
- ASP.NET Core MVC
- Razor Views
- Bootstrap
- OpenAI API
